# Automation Exercise UI Test Project

This project contains automated UI tests written in Java using Selenium WebDriver for the [https://automationexercise.com](https://automationexercise.com) website.

## 🔧 Technologies Used

- Java
- Selenium WebDriver
- WebDriverManager
- Maven
- IntelliJ IDEA

## ✅ Test Scenarios

- ✅ Valid Login Test
- ❌ Invalid Login Test
- 🆕 Create Account Test
- 🛒 Add to Cart Test
